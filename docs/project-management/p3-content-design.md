# P3 内容扩展设计

版本：v0.1 初稿

## 设计目标

P3 让玩家完成筑基前目标后，立刻看到新的可玩去处和成长方向。范围只到筑基初期继续玩，不做金丹完整投放，不新增地图、历练、战斗、炼丹或知识树系统。

玩家体验目标：

- 短期目标：发现并进入新区域，打通第一轮资源点和普通遭遇。
- 中期目标：完成 5-7 步事件链，拿到筑基初期功法、技能或丹方线索。
- 战斗感受：筑基初期普通战稳定可刷，精英战需要丹药或技能配置，Boss 是阶段检查点。
- 成长感受：练气技能仍能用，但筑基内容提供明确升级价值。

## 内容包概览

新增区域定为「雾隐溪谷」，定位在云岚仙城与天河渡口之间的筑基初期野外区域。

选择理由：

- 地理上可复用现有 `world_map.yaml` 的城市路线关系，不需要新地图系统。
- 主题承接黑水沼泽的毒雾、灵草和妖兽材料，同时引入筑基后的「道基稳固」目标。
- 资源、事件、怪物、丹方、技能都能走现有 YAML 配置和现有 UI。

建议 ID：

| 类型 | ID | 名称 | 用途 |
|---|---|---|---|
| wilderness_region | `mist_hidden_valley` | 雾隐溪谷 | 筑基初期新区域 |
| wilderness_location | `mist_creek_herb_garden` | 雾溪药圃 | 资源地点 |
| wilderness_location | `sealed_creek_ruins` | 封溪残阵 | 危险地点 |
| location | `mist_hidden_valley` | 雾隐溪谷 | 历练入口 |
| event prefix | `mist_hidden_valley__*` | - | 区域通用事件 |

## 核心循环

### Moment-to-Moment

- 行动：玩家在雾隐溪谷选择采药、调查、绕行或战斗。
- 反馈：事件文本明确标出「安全 / 消耗 / 精英 / 首领」风险；战斗掉落能看到新材料。
- 奖励：获得筑基材料、丹方线索、功法书或技能书。

### Session Loop

- 目标：从世界地图发现雾隐溪谷，完成一次历练并推进事件链。
- 张力：溪谷事件更耗法力，精英和 Boss 对筑基前玩家有明显失败风险。
- 结算：普通路线产出炼丹材料，事件链关键节点产出学习目标。

### Long-Term Loop

- 目标：稳定刷取材料，炼制筑基初期丹药，学习筑基技能 / 功法。
- 留存钩子：事件链结尾提示「封溪残阵深处仍有未解禁制」，作为后续金丹前内容伏笔，但 P3 不实现。

### 区域规格

| 字段 | 建议值 | 设计理由 |
|---|---:|---|
| danger | `[PLACEHOLDER] 3` | 高于黑水沼泽，表示筑基初期门槛。 |
| min_difficulty | `[PLACEHOLDER] 4` | 练气成型可以试探，但不应稳定碾压。 |
| max_difficulty | `[PLACEHOLDER] 8` | 给筑基初期普通、精英、Boss 留空间。 |

| near_city | `yunlan_city`, `tianhe_ferry` | 复用现有城市，不加城市节点。 |
| preview_rewards | 新灵草、新妖丹、筑基丹药材料 | 让玩家在地图上读懂收益。 |

### 子地点

`mist_creek_herb_garden` 是资源点。

- 目的：提供筑基初期丹方材料来源。
- 玩家决策：低风险采集，还是提高难度刷高品质材料。
- 输入：从世界地图进入，或在雾隐溪谷历练事件中触发。
- 输出：掉落新药材、灵果、少量灵石。
- 失败状态：空采或遭遇轻微法力损耗，不直接强制战斗。
- 边界：难度低于 4 时不掉最高品质材料。

`sealed_creek_ruins` 是危险点。

- 目的：承载精英 / Boss 与事件链终点。
- 玩家决策：是否在补给充足时深入残阵。
- 输入：完成事件链中段后在地图发现，或直接作为雾隐溪谷子地点显示。
- 输出：精英掉落、Boss 掉落、技能 / 功法书目标。
- 失败状态：战败按现有历练失败结算，不新增惩罚。
- 边界：Boss 事件 `once_per_lilian: true`，并要求 `difficulty_at_least: 6`。

| `mist_marten` | 雾溪灵鼬 | 普通 | `[PLACEHOLDER] 筑基普通 0.85x` | 快速普通战，掉基础材料。 |

## PM-302 区域事件链

事件链名：「雾溪封阵」

链路长度：6 步。全部走现有 `lilian_events.yaml` 决策、战斗、奖励字段。

| 步骤 | ID | 类型 | 玩家体验 | 结果 |
|---|---|---|---|---|
| 1 | `mist_creek_chain_tracks` | decision | 发现溪边阵纹，选择追查或先采药。 | 追查进入步骤 2；采药给少量材料后保留线索。 |
| 2 | `mist_creek_chain_fog_test` | hazard | 雾中试阵，消耗法力。 | 扣 `[PLACEHOLDER] 18% MP`，提示需要筑基后更稳。 |
| 3 | `mist_creek_chain_marten_battle` | battle | 普通妖兽偷袭。 | 掉基础材料。 |
| 4 | `mist_creek_chain_choice` | decision | 救被困采药人或直接破阵。 | 救人降低后续风险但少拿材料；破阵提高奖励并触发精英。 |
| 5 | `mist_creek_chain_guard_elite` | elite | 藤甲守卫现身。 | 掉核心材料或功法书线索。 |
| 6 | `mist_creek_chain_seal_core` | boss / reward | 封溪阵灵守住阵眼。 | Boss 后给明确下一目标：炼丹 / 学筑基功法。 |

关键选择：

- 救人：消耗 `items_LiaoShangDan` 或法力，后续 Boss 前获得恢复事件。适合稳健路线。
- 破阵：获得额外新矿材 / 新药材，但步骤 5 必定触发精英。适合输出路线。

手动复测要求：

- 新区域首轮历练至少能通过固定入口看到步骤 1 或步骤 2。
- 事件链不依赖纯随机撞齐；实现时可用地点事件池权重、一次性事件或测试专用入口保证复测。
- 完成步骤 6 后 UI 文案提示「去炼丹」「学习筑基功法」两条后续目标中的至少一条。

## PM-303 筑基初期技能 / 功法扩展

已有配置里存在不少筑基技能和功法。P3 不需要扩系统，优先把下面一小组作为雾隐溪谷奖励投放，并补齐缺失展示文案或掉落来源。

### 技能投放

| ID | 名称 | 倾向 | 获取 | 设计目标 |
|---|---|---|---|---|
| `ability.combat.blood_strike` | 震血拳 | 稳健 / 控制 | 精英掉落技能书或事件奖励 | 提供近战打断感，不替代所有输出技能。 |
| `ability.combat.five_phase_burst` | 五行轮爆 | 输出 / 清场 | Boss 或事件链结尾奖励 | 解决多敌压力，成本高于练气技能。 |

数值护栏：

- 震血拳基础伤害 `[PLACEHOLDER] 75` 保持低于五行轮爆，但附带 stagger 价值。
- 五行轮爆基础伤害 `[PLACEHOLDER] 95` 是 AoE 预算，不允许单体效率压过破空剑气太多。
- 两者继续用 `realm_balance.yaml` 的 foundation 预算校验。

### 功法投放

| ID | 名称 | 倾向 | 获取 | 设计目标 |
|---|---|---|---|---|
| `method.hunyuan.2` | 混元归一经·筑基篇 | 稳健 | 事件链完成后保底线索 | 扩法力和稳定性，服务普通玩家。 |
| `method.vajra.2` | 金刚不灭身·筑基篇 | 稳健 / 防御 | 封溪残阵精英掉落 | 对抗精英战，给低操作容错。 |

## 反馈与 UI 文案

不新增 UI 面板。只补现有入口和事件反馈：

- 世界地图区域描述：强调「筑基初期」「药圃」「残阵」。
- 历练地点 subtitle：说明资源点 / 危险点差异。
- 事件链完成文案：告诉玩家下一步是炼丹或学习筑基功法。
- 战斗失败提示：提示准备丹药、回头学习筑基技能，不只说失败。

## 验收与测试

自动检查：

- `tests/run_config_validation_tests.gd`
- `tests/run_world_map_tests.gd`
- `tests/run_lilian_tests.gd`
- `tests/run_simulation_tests.gd`
- `tests/run_dao_knowledge_tests.gd`
- `tests/run_balance_v1_tests.gd`

手动路线：

1. 完成筑基前目标或使用测试档进入筑基初期。
2. 从世界地图发现雾隐溪谷。
3. 进入雾溪药圃，完成一次资源采集。
4. 进入封溪残阵或雾隐溪谷事件池，推进「雾溪封阵」到精英节点。
5. 获得至少 1 个筑基功法 / 技能 / 丹方目标。
6. 回到洞府，确认背包、炼丹、功法 / 技能界面能展示新增内容。

P3 通过标准：

- 新区域、事件链、怪物、材料、丹方、技能 / 功法投放都能被现有系统表达。
- 玩家完成 P2 目标后，有「去雾隐溪谷」短期目标和「凑材料 / 学筑基内容」中期目标。
- 新内容不破坏 P0-P2 路线。
