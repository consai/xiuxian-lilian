## 路线核对

本轮采用 headless 回归和配置路径核对，没有做图形化长时点击游玩。

可复测手动路线：

1. 完成 P2 筑基前目标，或使用测试档进入筑基初期。
2. 从世界地图确认出现「雾隐溪谷」，并能进入历练。
3. 在雾隐溪谷完成一次资源采集，确认获得雾隐草或阵核碎片相关材料。
4. 推进「雾溪封阵」事件链：溪边阵纹 -> 雾阵试身 -> 守药灵鼬 -> 困阵采药人。
5. 选择「强行破阵」后触发藤甲守卫；提高难度后确认封溪阵灵作为 Boss 出现。
6. 结算后回到洞府，确认背包能显示新材料 / 丹药，炼丹状态包含清脉丹方和固本道基丹方。
7. 继续学习筑基知识或功法，确认角色派生属性有成长反馈。

## 结果

P3 内容可以通过现有地图、历练、战斗、炼丹、知识树系统表达，没有新增系统依赖。玩家在筑基前目标之后会获得「去雾隐溪谷」的短期目标，以及「凑材料炼丹 / 学筑基内容」的中期目标。
