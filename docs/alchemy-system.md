# 炼丹系统设计

## 1. 系统定位

炼丹是“将历练所得材料转化为可预期收益”的经营系统。

玩家使用固定丹方，不进行猜配方或自由投料。每次炼制的决策来自：

```text
丹方
+ 药材品质
+ 丹炉
+ 炼丹术等级
+ 炼制策略
= 无产物 / 废丹 / 下品 / 中品 / 上品 / 极品
```

设计目标：

- 固定丹方降低记忆负担，让玩家专注资源与风险管理。
- 炼制前展示结果概率，失败是玩家接受的风险，不是黑箱惩罚。
- 低品质药材始终有用途，高品质药材能明显提高成丹品质。
- 丹炉、炼丹术与策略提供不同成长方向，避免全部只是成功率加成。
- 炼丹与历练、突破、战斗消耗和交易形成资源循环。

## 2. 核心循环

```text
历练、采集、交易获得药材
-> 选择已掌握丹方
-> 选择符合丹方要求的药材批次
-> 选择丹炉与炼制策略
-> 查看六档结果概率、耗时与预期产量
-> 确认炼制并推进天数
-> 获得丹药或失败产物，增长炼丹术熟练
-> 使用、出售或为下一次炼制继续积累
```

一次炼制默认消耗 `shijian_rules.yaml` 中的炼丹基础耗时，并随境界倍率拉长。玩家可以连续炼制多炉，但每炉独立结算，方便展示结果和控制资源。

## 3. 固定丹方

丹方固定以下内容：

| 字段 | 作用 |
|---|---|
| 丹药 | 决定成功后产物及各品质效果 |
| 药材槽 | 固定材料族、数量和权重 |
| 难度 | 降低成丹分，决定推荐炼丹术等级 |
| 基础产量 | 一炉成丹时的基础数量 |
| 品质倾向 | 少数丹方可偏向稳定成丹或追求高品 |
| 掌握条件 | 丹方书、炼丹术等级、大道知识或剧情条件 |

固定丹方要求的是“材料族”，不是某一个具体物品 ID。例如聚气丹要求：

```text
灵草类 x2
聚灵果类 x1
```

同一材料族可以存在普通、良品、上等、珍品、天材五档物品。这样既保持丹方固定，也允许玩家选择药材品质。

### 首批丹方

| 丹方 | 难度 | 材料 | 基础产量 | 主要用途 |
|---|---:|---|---:|---|
| 回气丹 | 20 | 灵草类 x2 | 3 | 战斗外恢复气血 |
| 回灵丹 | 28 | 灵草类 x1、聚灵果类 x1 | 3 | 恢复法力、辅助修炼 |
| 疗伤丹 | 38 | 灵草类 x2、活血花类 x1 | 2 | 恢复气血、缩短伤势 |
| 聚气丹 | 45 | 灵草类 x2、聚灵果类 x1、妖丹类 x1 | 2 | 增加修为 |
| 破境丹 | 68 | 上等灵草类 x2、妖丹类 x1、凝神花类 x1 | 1 | 提高突破准备 |

高阶丹方继续沿用同一结构，但加入更高最低药材品质和更稀有的材料族，不增加投料槽复杂度。

## 4. 药材品质

药材品质使用五档：

| 品质 | 品质值 | 常见来源 |
|---|---:|---|
| 普通 | 1 | 初级采集、普通历练 |
| 良品 | 2 | 深入采集、稳定交易 |
| 上等 | 3 | 精英历练、药田培养 |
| 珍品 | 4 | 稀有事件、宗门交换 |
| 天材 | 5 | 高阶区域、重大机缘 |

每个药材槽拥有权重。主药通常为 `2`，辅药为 `1`。药材贡献为：

```text
药材分 = 加权平均品质值 × 8 - 丹方最低品质值 × 8
```

示例：某丹方最低品质为普通，投入两份良品主药和一份普通辅药：

```text
加权平均品质 = (2 × 2 + 1 × 1) / (2 + 1) = 1.67
药材分 = 1.67 × 8 - 1 × 8 = +5.3
```

药材品质主要抬高结果分，不直接保证成丹。高品质药材投入低阶丹方可以稳定获得高品丹药，但应受经济价值约束，通常不是最赚钱的选择。

### 与现有背包结构的兼容

当前背包以 `item_id -> count` 存储，不能让同一个物品 ID 同时拥有不同品质。首版采用“同材料族、不同物品 ID”：

```text
herb.spirit_grass.common
herb.spirit_grass.fine
herb.spirit_grass.superior
```

各物品通过 `alchemy.family` 指向同一材料族，通过 `alchemy.quality` 记录品质值。这样不需要把背包改造成带实例属性的复杂物品系统。

## 5. 丹炉

丹炉不是简单品质加成，而是由四项属性定义：

| 属性 | 作用 |
|---|---|
| 控火 `control` | 固定提高成丹分 |
| 炼化 `refinement` | 提高成丹数量或高品质结果收益 |
| 容错 `safety` | 降低无产物概率，部分无产物转为废丹 |
| 耐久 `durability` | 每炉消耗，归零后需修理 |

首版丹炉示例：

| 丹炉 | 控火 | 炼化 | 容错 | 耐久 | 定位 |
|---|---:|---:|---:|---:|---|
| 旧铜炉 | 0 | 0 | 0 | 20 | 初始炉 |
| 青石丹炉 | +5 | 0 | +8% | 40 | 稳定入门 |
| 赤火炉 | +3 | +15% | 0 | 35 | 提高产量 |
| 玄微炉 | +8 | +5% | +10% | 60 | 中期通用 |

`safety` 在结果为“无产物”时进行一次转换判定，成功则改为“废丹”。它不会把废丹直接变为成丹。

## 6. 炼丹术等级

炼丹术为独立生产等级，首版建议 `1-10` 级。

每级提供：

- `+3` 成丹分；
- 解锁更高难度丹方；
- 在特定等级解锁新策略；
- 降低低阶丹方难度或材料门槛；耗时由统一时间规则决定。

炼丹经验由丹方难度和实际结果共同决定：

```text
基础经验 = max(2, 丹方难度 - 炼丹术等级 × 4)
结果倍率：
无产物 0.7 / 废丹 0.9 / 下品 1.0 / 中品 1.1 / 上品 1.2 / 极品 1.4
```

炼制明显低于当前等级的丹方时，经验逐步衰减，避免用廉价材料无限刷等级。失败仍给予大部分经验，避免玩家因为一次失败同时损失材料、时间和成长。

大道知识中的“炼丹师”可作为炼丹术的解锁与专精入口，但不建议与炼丹术等级重复提供大量成丹分。大道知识更适合提供规则变化，例如降低丹炉耐久消耗、提高废丹回收价值或减少丹毒。

## 7. 炼制策略

策略决定玩家愿意承担什么代价。每次炼制只能选择一种。

| 策略 | 成丹分 | 波动 | 耗时 | 产量 | 特殊效果 |
|---|---:|---:|---:|---:|---|
| 循方炼制 | +0 | 标准 | +0 | +0 | 默认，无额外规则 |
| 稳守炉火 | +7 | 低 | +1 日 | +0 | 无产物时更容易转为废丹 |
| 催火成丹 | -6 | 高 | -1 日 | +25% | 成丹时有机会多产一枚 |
| 搏炼极品 | -5 | 极高 | +0 | +0 | 扩大高低两端概率，上品与极品概率提高 |
| 惜材慢炼 | +2 | 低 | +1 日 | -20% | 无产物或废丹时返还一份随机辅药 |

策略不应存在“永远最好”的选项：

- 材料昂贵时，稳守炉火或惜材慢炼更合理。
- 批量制作常用丹时，催火成丹节省时间。
- 低价材料充足、追求极品时，搏炼极品提供高风险机会。
- 循方炼制适合稳定日常生产。

## 8. 结果结算

### 8.1 成丹分

每炉先计算基础成丹分：

```text
基础成丹分 =
50
- 丹方难度
+ 药材分
+ 丹炉控火
+ 炼丹术等级 × 3
+ 策略成丹分
+ 属性辅助分
```

属性辅助分使用神识、悟性和灵根适配，但只提供小幅帮助，避免炼丹成为必须堆角色属性的系统：

```text
神识辅助 = clamp((神识 - 10) × 0.25, 0, 5)
悟性辅助 = clamp((悟性 - 10) × 0.20, 0, 4)
灵根适配 = 0 至 3，由丹方标签与灵根决定
属性辅助分上限 = 10
```

最终结果分：

```text
最终结果分 = 基础成丹分 + 随机波动
```

随机波动使用两个均匀随机数相加的三角分布，让结果更常靠近预期值，减少极端连续倒霉：

```text
标准波动：random(-12, 12) + random(-12, 12)
低波动：random(-7, 7) + random(-7, 7)
高波动：random(-17, 17) + random(-17, 17)
极高波动：random(-24, 24) + random(-24, 24)
```

### 8.2 六档结果

| 最终结果分 | 结果 |
|---:|---|
| `< 15` | 无产物 |
| `15-34` | 废丹 |
| `35-54` | 下品 |
| `55-69` | 中品 |
| `70-84` | 上品 |
| `>= 85` | 极品 |

丹炉容错仅在“无产物”结果后判定。废丹是一种可回收资源，可出售、完成低级委托，或在后续系统中提炼为少量药渣精华。

### 8.3 产量

```text
实际产量 = max(1, round(基础产量 × (1 + 丹炉炼化 + 策略产量修正)))
```

品质越高不额外增加数量，避免极品同时在强度和数量上双重获利。

### 8.4 品质效果

同一种丹药的不同品质使用独立物品 ID，便于兼容现有背包：

```text
pill.juqi.low
pill.juqi.medium
pill.juqi.high
pill.juqi.supreme
```

建议效果倍率：

| 品质 | 效果倍率 | 售价倍率 |
|---|---:|---:|
| 下品 | 0.80 | 0.65 |
| 中品 | 1.00 | 1.00 |
| 上品 | 1.25 | 1.60 |
| 极品 | 1.55 | 2.60 |

极品的价值应明显，但不应达到永久属性或突破必需品的程度。永久成长类丹药仍需服用次数、丹毒或边际递减限制。

## 9. 概率预览

炼制确认前必须展示六档概率。预览使用与实际结算完全相同的规则，通过枚举或固定种子模拟计算：

```text
无产物 4%  废丹 12%  下品 31%
中品 34%  上品 16%  极品 3%
```

同时展示关键变化来源：

```text
基础成丹分 61
丹方难度 -45
药材品质 +9
青石丹炉 +5
炼丹术 Lv.6 +18
稳守炉火 +7
```

玩家更换一份药材、丹炉或策略后，概率立即刷新。界面应突出变化量，例如“上品及以上：19% -> 27%”。

## 10. 示例结算

玩家使用聚气丹丹方：

- 丹方难度：`45`
- 药材分：`+6`
- 青石丹炉控火：`+5`
- 炼丹术等级：`6`，贡献 `+18`
- 策略：稳守炉火，贡献 `+7`
- 属性辅助分：`+4`

```text
基础成丹分 = 50 - 45 + 6 + 5 + 18 + 7 + 4 = 45
最终结果分 = 45 + 低波动
```

该配置主要产出下品与中品，失败风险较低。若改用“搏炼极品”，基础成丹分下降且波动扩大，平均收益未必提高，但会出现少量上品与极品机会。

## 11. 界面流程

炼丹界面采用单屏三栏：

### 左栏：丹方

- 按可炼制、已掌握、未掌握筛选。
- 显示用途、难度、基础产量和材料缺口。
- 默认选中最近使用的丹方。

### 中栏：投料与丹炉

- 固定显示丹方药材槽，不能增加、删除或替换材料族。
- 每个槽位可选择该材料族中拥有的品质。
- 提供“一键最低成本”和“一键最高品质”。
- 选择当前丹炉并显示耐久。

### 右栏：预测与确认

- 选择炼制策略。
- 显示六档结果概率。
- 显示预计耗时、材料消耗、产量和炼丹经验。
- 确认按钮文案为“开炉炼制（7日）”或对应的规则化时长。

结算时展示炉火过程的简短演出，然后显示本炉结果、炼丹经验、丹炉耐久变化和可继续炼制数量。首版不加入需要实时点击的火候小游戏，避免批量生产变成重复操作。

### 结算服务边界

建议新增独立 `LiandanService`，只负责：

- 校验丹方、药材、丹炉与策略；
- 生成概率预览；
- 使用传入随机数生成器结算单炉结果；
- 返回材料消耗、产物、耗时、经验和耐久变化。

`GameState` 负责应用结果、推进天数、记录日志与自动存档。UI 不直接修改背包或存档。

## 13. 经济与平衡护栏

- 同档药材直接出售价值应约为其日常炼丹期望产值的 `55%-70%`。
- 普通配置炼制同级丹方时，“成丹”总概率目标为 `65%-80%`。
- 达到推荐等级并使用匹配丹炉后，“无产物”概率应低于 `8%`。
- 越级炼丹可以尝试，但无产物与废丹合计不应低于 `45%`。
- 极品概率在常规配置下为 `1%-5%`，专门搏炼配置可提高到 `8%-15%`。
- 高品质药材不应被低阶丹方消耗后稳定套利。
- 失败损失应能通过 `1-3` 次同级普通历练恢复。
- 废丹必须有稳定消耗口，否则它只是额外的失败提示。

## 14. 首版落地范围

### P0：可玩的炼丹纵切片

1. 实现回气丹、回灵丹、疗伤丹、聚气丹四张丹方。
2. 实现普通、良品、上等三档药材。
3. 实现旧铜炉、青石丹炉两种丹炉。
4. 实现循方炼制、稳守炉火、搏炼极品三种策略。
5. 支持概率预览、单炉结算、推进天数、材料扣除和产物入包。
6. 增加固定种子测试，验证概率、材料守恒和存档兼容。

### P1：成长与经济

1. 接入炼丹术等级、丹方解锁与丹炉耐久。
2. 增加废丹回收、交易价格和炼丹委托。
3. 将聚气丹、破境丹接入修炼与突破准备。

### P2：专精与长期风险

1. 接入大道知识中的炼丹专精。
2. 增加丹毒、永久成长丹药服用限制和边际递减。
3. 增加药田、稀有丹方与特殊丹炉。

## 15. 验收指标

| 指标 | 目标 |
|---|---|
| 玩家能否在确认前理解失败原因 | 概率和贡献项完整可见 |
| 同一丹方是否存在至少三种合理策略 | 稳定、效率、搏品质均有使用场景 |
| 固定丹方是否仍有决策空间 | 药材品质、丹炉、策略均能显著改变分布 |
| 常规同级炼制成丹率 | `65%-80%` |
| 推荐配置无产物率 | `< 8%` |
| 常规极品率 | `1%-5%` |
| 失败恢复周期 | `1-3` 次普通历练 |
| 批量炼制是否产生重复操作负担 | 不要求逐炉实时火候操作 |

该系统的核心不是“猜中配方”，而是判断这炉丹值得投入多好的药材、多少时间，以及愿意承受多大的波动。
