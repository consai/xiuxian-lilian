# 道具品质体系与颜色

本文档约定道具、法宝、技能展示共用的品质档位。运行时统一通过 `res://scripts/enum/enum_quality.gd` 取显示名、颜色与兼容映射。

## 档位

| 数值 | 名称 | 定位 | 建议用途 |
| --- | --- | --- | --- |
| 1 | 普通 | 基础产物 | 新手资源、常规材料、低阶丹药 |
| 2 | 良品 | 稍优产物 | 较好材料、中品炼制结果、早期小惊喜 |
| 3 | 稀有 | 明显提升 | 上品材料、稀有丹药、核心掉落 |
| 4 | 极品 | 单品精品 | 极品丹、强力法宝、局部质变奖励 |
| 5 | 史诗 | 阶段核心 | 高阶传承、重要 build 件 |
| 6 | 传说 | 长线目标 | 传说丹药、传承法宝、稀缺奇遇奖励 |
| 7 | 仙品 | 超常规 | 后期仙缘、特殊活动、终局资源 |
| 8 | 神品 | 顶级唯一 | 神话级奖励、极少数终局追求 |

## 兼容输入

配置表推荐直接写 `quality: 1` 到 `quality: 8`。旧字段或外部配置可继续使用以下写法，运行时会统一映射：

| 输入 | 映射 |
| --- | --- |
| `普通`、`common`、`low`、`下品` | 普通 |
| `良品`、`uncommon`、`medium`、`中品` | 良品 |
| `稀有`、`rare`、`high`、`上品` | 稀有 |
| `极品`、`supreme` | 极品 |
| `史诗`、`epic` | 史诗 |
| `传说`、`legendary` | 传说 |
| `仙品`、`immortal` | 仙品 |
| `神品`、`mythic` | 神品 |
| `品质4` 或纯数字字符串 | 对应数值档位 |

## UI 规则

- 名称、悬浮提示、奖励提示使用 `EnumQuality.get_color(quality)`。
- 道具格从 `普通` 开始显示品质效果：底纹、内描边和左上品质珠都使用同一品质色。
- 底纹透明度使用 `EnumQuality.tint_alpha_from_label(label)`，低品质克制，高品质更明显。
- 背包、奖励结算、GM 面板中的法宝品质显示完整档位名，例如品质 4 显示为 `极品`，不再显示 `品质4`。
- 炼丹成丹品质仍保留 `EnumAlchemyQuality` 的结算语义，但展示到道具、奖励或背包时可映射到本体系。
